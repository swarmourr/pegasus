# Introduction
Runs a data flow corresponding to the tess_dense in `examples/tess_dense`
directory of a DECAF distribution (was tested on `workflow.isi.edu`).

# Requirements
- MPI distribution with mpirun in the PATH
- DECAF_PREFIX set to decaf install directory
- Pegasus installation and a Condorpool prefigured

make sure your PATH set in the sites.xml has the right `mpirun`.
